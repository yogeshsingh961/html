Block Elements

Def: Block elements are those elements which starts from a new line and takes complete width of the screen.

    Example:
    1. p tag
    2. headings from h1-h6
    3. div
    4. section
    5. ul
    6. ol
    7. nav
    8. form
    9. table
    10. aside
    11. article
    12. main

Inline Elements

Def: Inline elements are those elements which starts from the next point where the previous element has been ended.

    Example:
    1. image tag
    2. video tag
    3. audio
    4. span
    5. br
